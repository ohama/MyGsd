# .claude/ 디렉토리 유튜브 시리즈

Claude Code의 `.claude/` 디렉토리 활용법을 7개 에피소드로 다루는 유튜브 시리즈 스크립트입니다.

## 시리즈 개요

| EP | 제목 | 핵심 메시지 |
|----|------|-------------|
| 01 | [.claude/ 소개](ep01-intro.md) | .claude/ 폴더로 Claude 행동을 코드로 정의 |
| 02 | [Settings 설정](ep02-settings.md) | settings.json은 공유, local은 로컬 전용 |
| 03 | [Commands 슬래시 명령어](ep03-commands.md) | Markdown 파일 하나가 명령어 하나 |
| 04 | [Skills 자동 규칙](ep04-skills.md) | Skill은 Claude가 자동으로 따르는 규칙 |
| 05 | [Agents AI 서브프로세스](ep05-agents.md) | Agent는 독립 실행되는 전문화된 AI |
| 06 | [Hooks 이벤트 스크립트](ep06-hooks.md) | Hook은 이벤트에 반응하는 Node.js 스크립트 |
| 07 | [실전 활용 사례](ep07-practical.md) | 배운 것을 실제 프로젝트에 적용 |

## 대상 시청자

- Claude Code 사용자
- AI 코딩 도구에 관심 있는 개발자
- 반복 작업을 자동화하고 싶은 분

## 스크립트 구성

각 에피소드는 다음 구조로 작성되었습니다:

1. **메타 정보**: 영상 길이, 대상, 핵심 메시지
2. **인트로**: 30초 내외의 주제 소개
3. **본론**: 3-5개 섹션으로 나눈 핵심 내용
4. **아웃트로**: 요약 및 다음 영상 예고
5. **편집 노트**: 영상 제작 시 참고 사항
