# Claude Code 설정 가이드

Claude Code의 `.claude/` 디렉토리를 활용하여 프로젝트별 설정, 명령어, 스킬, 에이전트, 훅을 정의하는 방법을 다룹니다.

실제 프로젝트(LangTutorial)에서 `.claude/`를 어떻게 활용했는지 참고하여, 각 구성요소의 역할과 작성법을 단계별로 설명합니다.

## 대상 독자

- Claude Code를 사용하는 개발자
- 프로젝트별로 Claude의 동작을 커스터마이즈하고 싶은 사람
- GSD 프레임워크의 내부 구조를 이해하고 싶은 사람
